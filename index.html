CalibrePHP

This is a simple HTML and OPDS web server to access a database created by the Calibre application. CalibrePHP was written using CakePHP.

Requirements

The master branch has the following requirements:

  • HTTP Server. For example: Apache with mod_rewrite
  • PHP 5.3.0 or greater.
  • PHP Sqlite 3 support.
  • ImageMagick.

Other Requirements:

  • Calibre library directory and sub-directories need to be readble and executable by apache.

Installation

  • Copy/Clone the repository to your webserver.
  • Copy/Rename app/Config/database.php.default to app/Config/database.php.
    • change $default['database'] to match the location of your calibre database.
    • The path either needs to be the full path from '/', or relative from the app/webroot/ directory.
  • Copy/Rename app/Config/settings.php.default to app/Config/settings.php.
    • change the e-mails to what you want, or set as an empty array() to disable.
  • Copy/Rename app/Config/core.php.default to app/Config/core.php.
    • change Security.salt from the default.
    • change Security.cipherSeed from the default.
    • change debug to the desired level.
